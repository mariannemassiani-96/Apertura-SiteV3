# Apertura di Corsica

Site vitrine premium Next.js (App Router) pour Apertura di Corsica.

## Installation
1. `npm install`
2. `npm run dev`
3. Ouvrir `http://localhost:3000`
4. `npm run build` pour la production
5. Déployer sur Vercel (import du repo)
